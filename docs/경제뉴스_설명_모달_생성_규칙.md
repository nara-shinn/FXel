# FXel 경제뉴스 설명 모달 — 콘텐츠 생성 규칙

## 1. 개요

FXel의 중앙 카드 모달은 초보 투자자가 경제 뉴스를 빠르게 이해하도록 돕는 Flip 카드 UI다.
앞면은 뉴스 요약(타임라인 노트), 뒷면은 사고 확장(인사이트)을 보여준다.

콘텐츠 생성 시 지켜야 할 3가지 대원칙:

- 단순 정의가 아니라 **현재 시장 상황과 연결된 맥락**으로 설명한다.
- 모든 내용은 **실제 뉴스 기사 기반**이며, AI가 임의로 추론하거나 사실을 만들면 안 된다. (최근 기사 우선)
- 설명은 **초보 투자자 기준**으로 쉽고 직관적이어야 한다.

## 2. 필드 매핑표

| 필드명 | 위치 | 내용 |
|---|---|---|
| `term` | 앞면 헤더 | 키워드 배지 |
| `headline` | 앞면 헤더 | 한 줄 요약 헤드라인 (해요체) |
| `timelineNode1_Concept` | 앞면 Node 1 | 쉬운 개념 설명 |
| `timelineNode2_News` | 앞면 Node 2 | 핵심 뉴스 팩트 (리스트) |
| `timelineNode3_ImpactChips` | 앞면 Node 3 | 시장 영향 키워드 (Chip) |
| `sources` | 앞면 Footer | 출처 기사 (가로 스크롤) |
| `ctaTriggerButtonText` | 앞면 버튼 | 뒷면으로 flip 트리거 |
| `question` | 뒷면 중앙 | 소크라테스식 질문 |
| `insight` | 뒷면 하단 | 질문에 대한 인사이트 (탭/호버 시 노출) |

## 3. 앞면 UI — 타임라인 노트 구조

1. **Header** — `term`(키워드 배지) + `headline`(해요체 한 줄 요약)
2. **Node 1 (원인/개념)** — 연한 배경 블록 + 빈 원형 노드. 뉴스 읽기 전 개념부터 잡아주는 `timelineNode1_Concept`
3. **Node 2 (현상/뉴스)** — 채워진 원형 노드 + 세로 점선. 기사에서 추출한 핵심 팩트를 순차 리스트로 (`timelineNode2_News`)
4. **Node 3 (결과/영향)** — 가로 배열 Chip. 명사형 시장 영향 키워드 (`timelineNode3_ImpactChips`)
5. **Footer** — 가로 스크롤 출처 카드 (`sources`), 모달이 세로로 무한정 길어지는 것 방지
6. **CTA 버튼** — 우측 하단, 뒷면으로 flip

## 4. 뒷면 UI — 인사이트 플립

1. **중앙** — 큰 따옴표 스타일 + 능동적 사고를 유도하는 `question`
2. **하단** — 탭/호버 시 서서히 드러나는 `insight` (초등학생 눈높이)
3. **버튼** — "다시 뉴스 확인하기" → 앞면으로 복귀 (RotateY 0deg)

## 5. 필드별 생성 규칙

### headline (헤드라인)
- 반드시 **해요체**
- 기사 제목이 아니라 "설명을 읽는 느낌"이어야 함

| 좋은 예 | 나쁜 예 |
|---|---|
| 정부가 환율 안정을 위해 나설 수 있다고 밝혔어요 | 정부 발언으로 환율 움직임에 영향 |
| 중동 긴장이 커지며 달러를 찾는 사람이 늘었어요 | 중동 긴장으로 환율 상승 |
| 미국 경제가 예상보다 좋아 달러 가치가 올랐어요 | 미국 경제 호조에 달러 강세 |

### timelineNode1_Concept (쉬운 설명)
- 경제 초보자 기준, 전문 용어 최소화
- 어려운 용어가 나오면 반드시 풀어서 설명

예시: "달러인덱스가 오르면 달러 가치가 강해졌다는 뜻입니다. 보통 원화는 상대적으로 약해질 가능성이 있습니다."

### timelineNode2_News (무슨 일이 생겼나요?)
- 기사 내용 기반, **AI 해석만으로 작성 금지** (추론 금지)
- 각 문장 35자 이내, 개조식, 중요도 순 정렬
- 최근 뉴스 기사 + 현재 시장 상황과 연결

> ⚠️ 핵심 포인트 개수(3개 vs 예시 4개) 불일치 — 실제 운영 기준 확인 필요

예시:
```json
[
  "정부가 환율 급등을 우려한다고 밝혔어요",
  "시장 안정 의지를 공개적으로 언급했어요",
  "투자자들이 개입 가능성을 주목하고 있어요",
  "환율 상승 속도가 둔화될 수 있어요"
]
```

### timelineNode3_ImpactChips (시장 영향)
- 사용자의 실제 투자 판단에 참고가 되도록 작성
- 문장형이 아닌 **명사형 Chip**으로 처리 (인지부하 감소)

예시:
```json
[
  "달러 강세 → 원화 약세 가능성",
  "수입 물가 상승 압력",
  "외국인 자금 흐름 변화 가능성"
]
```

### sources (출처 기사)
- 설명에 사용된 핵심 내용은 기사에서 확인 가능한 정보여야 함
- 투명성 확보를 위해 반드시 함께 제공

### question / insight (생각해볼 지점)
- `question`: 초등학생도 이해 가능한 수준, 사고를 유도하는 질문. **정답을 직접 제공하지 않음**
- `insight`: 기사와 직접 관련된 내용만 사용

형식 예시:
```json
{
  "question": "정부가 실제 개입하지 않아도 왜 환율이 움직일까요?",
  "insight": "시장은 미래에 일어날 가능성에도 반응해요."
}
```
