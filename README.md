# Fixel — 숫자 뒤의 경제를 읽다

> **나의 환율 타이밍 가이드**  
> 단순한 환율 앱이 아닌, 거시경제 맥락을 읽는 의사결정 도구

---

## 서비스 소개

**Fixel(FXel)** 은 FX(환율) + Excel(성장하다/능가하다)의 합성어입니다.  
흐릿했던 글로벌 경제 시야를 픽셀 단위까지 선명하게(High-Definition) 만들어 준다는 시각적 은유를 담고 있습니다.

경제 지식이 얕은 사용자도 환율이 *왜* 움직이는지 이해하고, 환전·해외 투자·송금의 **타이밍**을 스스로 판단할 수 있도록 돕는 반응형 웹앱입니다.

---

## 핵심 가치

| | |
|---|---|
| **⏱ 타이밍** | 숫자만 보고 진입하면 늦다. 환율이 움직인 진짜 이유를 알면 완벽한 타이밍이 보인다. |
| **💪 결정 근육** | 정직한 데이터와 직관적인 맥락으로 사용자가 스스로 판단하는 감각을 키운다. |
| **🛡 금융 방어** | 복잡한 거시경제 이슈를 단 하나의 명확한 인과관계로 풀어낸다. |

---

## 주요 기능

### 1. 실시간 환율 조회
- USD · JPY · EUR · CNY 멀티통화 지원
- 전일 대비 변동폭 및 상승/하락 통계 시각화
- 최근 3개월 포지션 언어화 배지 ("최근 3개월 중 중간 수준이에요")

### 2. 환율 변동 추이 차트
- Robinhood 스타일 터치 스크러빙 — 손가락으로 차트를 쓸면 상단 수치가 실시간으로 변동
- 손을 떼면 현재 시점으로 스냅백
- 그리드·Y축 제거로 극단적으로 단순화한 AreaChart
- 인사이트 도트(Insight Dot) — 변동 분기점 클릭 시 이슈 요약 연결

### 3. 환율 영향 이슈 키워드
- 거시경제 이슈 키워드 칩(Chip) 형태로 표시
- 탭 시 3줄 요약 바텀 시트 노출
  - **헤드라인 선제 구조** (TDS 벤치마킹): 대형 한 줄 요약 → 태그 칩 → Q/A 카드 → 생각해볼 지점
- 플래시카드 플립 인터랙션 ("생각해볼 지점" 뒷면)
- 도움됨/별로예요 피드백 (LocalStorage 영속)

### 4. 목표 환율 알림
- 프리셋 3종: 최근 평균 이하 / 3개월 최저점 근처 / 1주일 최저점 근처
- 직접 구간 설정 지원
- LocalStorage 저장 (로그인 없이 사용 가능)
- 구간 진입 시 인앱 Toast 알림

### 5. 환율 계산기
- 네이버 증권 스타일 UI (Figma 설계 기반)
- 기본값: 대한민국 KRW 1,000원 → 미국 USD
- 국기 + 국가명 + 통화 코드 선택 드롭다운
- 더블바(≡) 구분선 · 교체 버튼 · 결과 복사

| Language | TypeScript |

```bash

npm install

npm run dev
```

## 설계 철학

- **단순 정보 제공 X, 맥락 제공 O** — 환율 수치보다 "왜 이렇게 움직였는가"를 먼저 전달
- **Robinhood 인터랙션** — 터치 드래그 스크러빙, 그리드 제거, 스냅백 애니메이션
- **TDS(Toss Design System) 타이포그래피** — 헤드라인 선제 구조, 명확한 계층 체계
- **Naver Securities 계산기** — Figma 디자인 스펙 직접 구현
